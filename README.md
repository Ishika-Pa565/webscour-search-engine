Web Search Engine Project

Milestone 1
Implemented a simple web crawler to fetch web pages.

Milestone 2
Developed a distributed web crawler using RabbitMQ with producer and multiple workers.

Milestone 3
Built an indexer to process HTML files, extract text, and create an inverted index with TF-IDF.

Milestone 4
Created a search engine using FastAPI and a simple UI to search documents.

Features
- Distributed crawling
- HTML parsing
- Inverted index
- TF-IDF ranking
- FastAPI search API
- Simple UI

How to Run

1. Run crawler (producer & worker)
2. Run indexer
3. Run search API
4. Open UI in browser
